Q: How to deploy a new apps?

1. Create a plan on what you will build
2. In commandline, you can start a new app "python manage.py startapp "
3. Working on models.py for data structure
4. Write data processes in views.py
5. Create templates in /templates/ and linked with render in views.py
6. Setup urls (app) and also edit "kargo/urls.py" for connection
7. run "python manage.py makemigration or migrate to implement models updates"
8. Done!
